Zu beachten: Dies ist eine Demoversion, die einen ersten Überblick über mögliche Anwendungsfälle der ohsome-Tools des Heidelberger Instituts für Geoinformationstechnologie geben soll.

Über die Demo

Diese Demoversion soll mit den verschiedenen Fallbeispielen umreißen, wie die geplante Anwendung aufgebaut ist und welche Möglichkeiten zur Abschätzung der Qualität enthalten sein werden.

Erste Eindrücke der Geodatenanalyse mit Werkzeugen der ohsome API sollen hierdurch gewonnen werden.

Fallbeispiele, die mit den ohsome-Tools untersucht werden, sind die zeitliche Entwicklung von Daten in OpenStreetMap, die Vollständigkeit und Aktualität sowie Vergleiche mit Referenzdatensätzen, Reisezeiten oder der logischen Konsistenz von Höhenwerten.

Wie funktioniert die Demoversion?

Im ersten Schritt soll eine vordefinierte Region über die interaktive Weltkarte gewählt werden. Für jede Region sind verschiedene Prüfszenarien hinterlegt.

Im nächsten Schritt soll der gewünschte Report aus einem Dropdown-Menü ausgewählt werden. Derzeit kann zwischen fünf verschiedenen Reports gewählt werden, die jeweils eine Abschätzung eines Qualitätsmerkmals enthalten. 

Im letzten Schritt soll der gewünschte Report angefordert werden, die Ergebnisse werden unterhalb des Dropdown-Menüs gelistet.

Welche Informationen umfasst die Demoversion?

Die folgenden Reports stehen zur Verfügung:

Vollständigkeit, Thematische Genauigkeit, Aktualität, Logische Konsistenz & Positionsgenauigkeit

Für die Regionen: Libanon (Beirut), Israel (Tel Aviv), Ukraine (Kyiv), Vietnam (Hanoi, rural), Kenia (Kiambu), Deutschland (Brandenburg, Rhein-Main) & Irland
