Please note, this is a demo which is supposed to provide a first overview of potential use cases for OpenStreetMap related products of the Heidelberg Institute for Geoinformation Technology.

About this demo

In the context of this demo, various use cases are included, which are intended to help with a visualization of a potential application.

Here you can get a first impression of the possibilities for spatial data analysis using tools such as the ohsome API or several quality indicators.

These use cases investigate aspects such as temporal evolution of OpenStreetMap, completeness and currentness as well as comparisons with reference data sets, travel times or logical consistency of height values.

How does this demo work?

In the first step of using the demo you can choose a location by selecting it in the interactive map.

There are certain use cases for each of these locations. The pre-selection is therefore made on the basis of a spatial parameter.

Then you can select the report you want to get information on for this location in the dropdown menu.

At last, you simply have to click on the button and your requested analysis results will be listed below the dropdown menu!

What information is included in the demo?

The following reports are available so far:

Completeness, Thematic Accuracy, Temporal Accuracy, Logical Consistency & Positional Accuracy

For the regions: Lebanon (Beirut), Israel (Tel Aviv), Ukraine (Kyiv), Vietnam (Hanoi, rural), Kenya (Kiambu), Germany (Brandenburg, Rhein-Main) & Ireland
